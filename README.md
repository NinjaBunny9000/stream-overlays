## Install
```
# change your-app to the name of your project
cd your-app
git remote remove origin
# edit the package.json, then continue on
npm install
npm run dev
```

## Build for prod
```
npm run build
```

## Quick Note
This boilerplate assumes your app code lives in /src.
main.js is the entry point for your app.
